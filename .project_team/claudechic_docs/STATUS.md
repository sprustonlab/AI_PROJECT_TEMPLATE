**Status:** SETUP

## Vision (Approved)

Document the core systems in `submodules/claudechic/` — hints, advance checks, rules, phases, workflows, and chicsessions — that are used by the template. Create `.claude/rules/` files so agents automatically get context when working with these systems.

## Deliverables

1. **Documentation** — Docs explaining each claudechic system: what it does, how it's configured, how it works at runtime, how to extend it
2. **Claude Code Rules** — `.claude/rules/` markdown files that auto-inject context into agents when they work with framework files
3. **Coverage of all 6 systems:** Hints, Advance Checks, Rules, Phases, Workflows, Chicsessions
