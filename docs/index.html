AI Project Template

Scaffold a Claude Code project with environments, guardrails, and multi-agent workflows.

Linux / macOS
Windows

Run this in your terminal:

curl -fsSL https://sprustonlab.github.io/AI_PROJECT_TEMPLATE/install.sh | bash -s my-project

Replace my-project with your project name. The script installs pixi if needed, runs Copier, and sets up environments.

Run this in PowerShell:

iwr -useb https://sprustonlab.github.io/AI_PROJECT_TEMPLATE/install.ps1 | iex

The script will prompt for your project name and install location.

Or download and run manually:

Download install.ps1
Using WSL? Use the Linux/macOS command instead.

Already have pixi?

With pipx or pixi exec:

pixi exec --spec "copier>=9,<10" -- copier copy https://github.com/sprustonlab/AI_PROJECT_TEMPLATE my-project

Then install environments:

cd my-project && pixi install
